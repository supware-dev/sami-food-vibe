Use [Google Maps for Flutter](https://pub.dev/packages/google_maps_flutter)

Supported Languages: Arabic, English, and French

Select Language on start

Setting to change language

Order Has an Id and Items
* Products (Estimated Preparation Time, Prices)
* Contact (By Phone)
* Delivery Options (Takeaway or Delivery (Min Per Price Or Items Count at the Product Level) (To Location (Prompt For Location Access)))
* Cart (Clear, Set Count Per Item Or Remove, Add More Items (Navigates To Products), Checkout)
* Checkout
* Order Tracking (Accepted, Preparing, ...)
* Orders History (Has Reorder, Feedback)
* Order Page from Orders History (Items, Delivery Options, Total Price)
* Order Customization (Ingredients and Count)
* Review
* Feedback (Post-Order) (Like/Dislike, Predefined Comments, Comment)
* Profile Page (To Set Name and Phone Number)
